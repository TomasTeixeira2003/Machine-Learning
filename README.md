# IST Machine Learning Portfolio (2023/2024)

This repository contains the complete set of four homework assignments developed for the **Machine Learning** (Aprendizagem) course at LEIC, Instituto Superior Técnico. Each project combines theoretical mathematical foundations with practical Python implementations using `scikit-learn`.

---

## Homework I: Decision Trees & Feature Selection
**Focus:** Supervised Learning and Information Theory.
* **Theoretical:** Calculations of Shannon Entropy, Information Gain, and Laplace smoothing for decision tree induction.
* **Practical:** Discriminative power analysis using `f_classif` on orthopedic data. Assessment of model generalization (overfitting vs. underfitting) through tree depth tuning.



---

## Homework II: Probabilistic & Instance-Based Learning
**Focus:** Classification Paradigms and Statistical Validation.
* **Theoretical:** Naive Bayes probability estimation and distance-weighted k-NN regression using Hamming distance.
* **Practical:** Statistical hypothesis testing (t-tests) to compare k-NN and Gaussian Naive Bayes. Analysis of cumulative confusion matrices through 10-fold stratified cross-validation.



---

## Homework III: Neural Networks & Regression
**Focus:** Non-linear Regression and Optimization.
* **Theoretical:** Radial Basis Function (RBF) networks and manual Backpropagation gradients for Multi-Layer Perceptrons (MLP).
* **Practical:** Wine quality estimation using MLP regressors. Comparative study of **Early Stopping** versus fixed iteration counts and residue distribution analysis.



---

## Homework IV: Unsupervised Learning & PCA
**Focus:** Clustering and Dimensionality Reduction.
* **Theoretical:** Bayesian clustering with Bernoulli/Gaussian mixtures and Silhouette coefficient manual calculations.
* **Practical:** k-means clustering optimization ($k=2$ to $5$) and **Principal Component Analysis (PCA)** for 2D data projection and feature relevance ranking.
